# PG Scanner

This is a real time AI physicans assistant when it comes to diagnosing Pyrdoerma Gangrenosum (PG). Just hold the skin ulcer in the camera frame from about 6 inches away and the model will provide you with a confidence percentage. You are then able to export the results into a PDF that will have an image of the condition and the confidence level. It will also store some basic paitent data into a local database for charting later.

**Project Type:** Google Teachable Image Analyzer Web-app
**Status:** Beta / Proof of Concept
**Created by:** Capstone Group 8: Ben Wilson, Prisha Jitaly, Blake McCullough, Haydon Spaulding

- Live webcam feed classification
- Confidence level display
- PDF export of results
- Local database to store basic paitent data

## How it was made
### Tech used: HTML, JavaScript (JS), CSS
This project was made using an html file to create a layout for the UI and then JS to connect the buttons and different pages. When you click on a button within the web app there is a JS function that is called to execute the button functionallity. In order to store the paitent data and pdf image we used IndexedDB which is a API that allows for managing a NoSQL database.
